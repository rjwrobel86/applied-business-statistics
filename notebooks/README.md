# Notebooks

This folder contains Jupyter notebooks for topics in statistics, econometrics, data analysis, and related methods in Python.

The notebooks are grouped by subject area and include both classroom examples and broader demonstrations. Some are introductory, while others go beyond the core course material.

## Regression

The `regression` folder is divided into:

- `fundamentals`
- `generalized-linear-models`
- `regularization`
- `panel-and-hierarchical`
- `advanced-regression`
- `bayesian`

These notebooks include material on linear regression, logistic and count models, regularized regression, multilevel models, quantile and robust regression, survival analysis, and Bayesian approaches.

## Notes

The notebooks include both core course materials and additional examples that go beyond the main course sequence. Some are polished instructional materials, while others are working notes or in-progress explorations.
