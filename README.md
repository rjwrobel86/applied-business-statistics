# Applied Business Statistics

Materials for Applied Business Statistics (BUSN 5760) at Webster University.

This repository includes datasets, Jupyter notebooks, assignments, study guides, slides, practice materials, and reference resources used throughout the course. The notebooks cover core topics in applied statistics along with additional material in econometrics, forecasting, machine learning, and data analysis in Python.

## Repository Structure

### `notebooks`
Jupyter notebooks organized by topic, including descriptive statistics, probability, hypothesis testing, regression, causal inference, diagnostics, simulation, time series, machine learning, natural language processing, visualization, and related areas.

### `datasets`
Datasets used in notebook examples, classroom demonstrations, assignments, and practice exercises.

### `course-materials`
Supporting instructional materials, including the syllabus, homework assignments, study guides, practice problems, slides, Excel materials, reference resources, and statistical tables.

## Topics

The repository includes material on descriptive statistics, probability, sampling, confidence intervals, hypothesis testing, ANOVA, chi-square tests, regression, generalized linear models, causal inference, simulation, time series analysis, forecasting, machine learning, and data visualization.

## Notes

Most materials are organized for teaching and learning, though some notebooks are exploratory, extended, or still in progress.
